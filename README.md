# 🤖 Бот «Личный Ассистент Дня» — MVP

Что умеет:
- ✅ Задачи и напоминания свободной фразой: «напомни завтра в 9 позвонить маме»
- 📸 Фото холодильника → список продуктов + 2–3 рецепта
- 🍳 Список продуктов текстом → рецепты
- ☀️ Утренний брифинг: погода + список задач в заданное время

---

## Шаг 1. Получи ключи (нужно 2 обязательных + 1 по желанию)

1. **Токен бота** — в Telegram найди `@BotFather` → `/newbot` → придумай имя → скопируй токен вида `123456:ABC-DEF...`
2. **Ключ AgentRouter** — [agentrouter.org](https://agentrouter.org) → войди через GitHub → «Получить API Key» → создай ключ. Новым пользователям дают бесплатные кредиты — карта не нужна. Бот ходит на `https://agentrouter.org/v1` с моделью `gpt-5`
3. *(необязательно)* **Ключ погоды** — [openweathermap.org](https://openweathermap.org/api) → бесплатная регистрация → API key. Без него всё работает, просто в брифинге не будет погоды.

## Шаг 2. Настрой .env

Скопируй `.env.example` в файл с именем `.env` и вставь свои ключи:

```
BOT_TOKEN=123456:ABC-DEF...
OPENAI_API_KEY=sk-...
OPENWEATHER_API_KEY=...
```

## Шаг 3. Запуск

### Локально (проверить на своём ПК)

```bash
pip install -r requirements.txt
python3 main.py
```

### На хостинге (ядро node.js + python3)

1. Залей все файлы папки на сервер (включая заполненный `.env`)
2. В консоли сервера: `pip install -r requirements.txt` (или `pip3`)
3. Команда запуска: `python3 main.py`

Бот работает в режиме polling — никаких портов, доменов и webhook настраивать не нужно.

## Шаг 4. Проверь

1. Напиши боту `/start`
2. Настрой: `/city Москва` и `/time 07:30`
3. Попробуй:
   - «напомни через 2 минуты попить воды» → через 2 минуты придёт напоминание
   - «курица, картошка, сметана» → бот предложит рецепты
   - отправь фото холодильника → бот распознает продукты и предложит рецепты
   - `/digest` → покажет утренний брифинг прямо сейчас (не ждать утра)

## Команды бота

| Команда | Что делает |
| --- | --- |
| `/start` | приветствие и регистрация |
| `/city Москва` | город для погоды |
| `/time 07:30` | во сколько присылать брифинг |
| `/tasks` | список открытых задач |
| `/done 3` | закрыть задачу №3 |
| `/digest` | брифинг прямо сейчас |
| `/help` | справка |

## Устройство

- `main.py` — весь код бота (один файл, с комментариями)
- `assistant.db` — база SQLite, создаётся автоматически при первом запуске
- Понимание фраз, рецепты и распознавание фото — через AgentRouter (модель gpt-5, OpenAI-совместимый API)

## Частые проблемы

- **Бот не отвечает** — проверь BOT_TOKEN в `.env` и что бот запущен (в логах есть «Бот запущен»)
- **Ошибка 401 от LLM** — неверный AGENTROUTER_API_KEY или закончились кредиты — проверь баланс в личном кабинете agentrouter.org
- **AgentRouter не отвечает / таймаут** — поменяй в `.env` адрес на зеркало: `LLM_BASE_URL=https://co.agentrouter.org/v1`
- **Отвечает медленно** — gpt-5 «думает» перед ответом; поставь в `.env` более быструю модель: `LLM_MODEL=gpt-5-mini`
- **Нет погоды** — ключ OpenWeatherMap активируется до пары часов после регистрации
- **Напоминания приходят не в то время** — проверь TZ_NAME в `.env` (например, Europe/Moscow)
